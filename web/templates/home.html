API

Calls:

/api/

Returns all centres

/api/[clientid]

Returns a single centre object

/api/[lng]/[lat]

Returns the 10 nearest centers to the lat/lng

Formats

Add ?format=[json|xml|yaml|kml] to any query to change the output format.

Add ?format=json&callback=[callback] for JSONP
